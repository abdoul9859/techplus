Guide de Démarrage Rapide

Bienvenue ! Suivez ces étapes pour configurer votre application TechPlus.

Comptes par défaut :
  • Admin : admin / admin123
  • Utilisateur : user / user123
  1. Accédez à http://localhost:8000
  2. Connectez-vous avec le compte admin
  3. Changez immédiatement le mot de passe par défaut
  4. Explorez le dashboard pour vous familiariser

  1. Allez dans Paramètres (icône engrenage)
  2. Remplissez les Informations de l'entreprise :
    • Nom de l'entreprise
    • Adresse complète
    • Téléphone et email
    • Logo (optionnel)
  3. Configurez les Paramètres fiscaux :
    • Devise : XOF (Franc CFA) par défaut
    • Taux de TVA : 18% par défaut
    • Numéro RC et NINEA
  4. Enregistrez les modifications

  1. Accédez au menu Clients
  2. Cliquez sur "Nouveau Client"
  3. Remplissez les informations :
    • Nom et prénom
    • Téléphone et email
    • Adresse de livraison
    • Type de client (particulier/entreprise)
  4. Enregistrez le client
  5. Répétez pour vos clients principaux

  1. Allez dans Produits
  2. Cliquez sur "Nouveau Produit"
  3. Remplissez les informations de base :
    • Nom du produit
    • Catégorie (Smartphones, Ordinateurs, etc.)
    • Prix d'achat et de vente
    • Quantité en stock
  4. Pour les produits avec variantes (ex: couleurs) :
    • Ajoutez des variantes avec IMEI/numéros de série
    • Définissez l'état (neuf, occasion, venant)
  5. Les codes-barres sont générés automatiquement
  6. Enregistrez le produit

  1. Accédez au menu Devis
  2. Cliquez sur "Nouveau Devis"
  3. Sélectionnez le client
  4. Ajoutez les produits :
    • Recherchez par nom ou scannez le code-barres
    • Définissez les quantités
    • Les prix se remplissent automatiquement
  5. Vérifiez le total et la TVA
  6. Enregistrez et imprimez le devis

  1. Dans la liste des devis, trouvez votre devis
  2. Cliquez sur "Convertir en facture"
  3. Vérifiez les informations
  4. Sélectionnez les numéros de série/IMEI si applicable
  5. Choisissez le mode de paiement
  6. Validez la facture
  7. Le stock se met à jour automatiquement
  8. Imprimez la facture pour le client
Conseil : Commencez par quelques produits et clients pour vous familiariser, puis étoffez progressivement votre catalogue.

Dépannage & FAQ

Impossible de se connecter :
  • Vérifiez que vous utilisez les bons identifiants
  • Comptes par défaut : admin/admin123 ou user/user123
  • Assurez-vous que Caps Lock n'est pas activé
  • Essayez de vider le cache du navigateur
Session expirée :
  • Reconnectez-vous simplement
  • Les sessions expirent après 24h d'inactivité

Les filtres sur la page produits ne marchent pas :
  • Actualisez la page (F5)
  • Vérifiez que JavaScript est activé
  • Essayez de vider les filtres et recommencer
  • Si le problème persiste, contactez le support
Recherche ne trouve pas de résultats :
  • Vérifiez l'orthographe
  • Essayez avec moins de mots-clés
  • Utilisez la recherche par catégorie

Erreurs dans les totaux de factures :
  • Vérifiez la configuration du taux de TVA dans les paramètres
  • Assurez-vous que les prix produits sont corrects
  • Contrôlez les remises appliquées
Stock incorrect :
  • Vérifiez l'historique des mouvements de stock
  • Contrôlez les factures récentes
  • Effectuez un inventaire si nécessaire

Facture ne s'imprime pas correctement :
  • Utilisez le format PDF pour l'impression
  • Vérifiez les paramètres d'impression (format A4)
  • Assurez-vous que l'imprimante est bien connectée
Codes-barres illisibles :
  • Imprimez en haute qualité
  • Utilisez du papier blanc de bonne qualité
  • Vérifiez que l'encre n'est pas faible

Application lente :
  • Videz le cache du navigateur (Ctrl+Shift+Del)
  • Fermez les onglets inutiles
  • Redémarrez le navigateur
  • Vérifiez votre connexion internet
Chargement des pages long :
  • Utilisez les filtres pour limiter les résultats
  • Évitez d'afficher trop de produits à la fois
  • Redémarrez l'application si nécessaire
